# Investo – Startup Idea & Investor Network Platform

Investo is a full-stack MERN application designed to connect entrepreneurs and investors in one place. Entrepreneurs can publish their startup ideas, while investors can explore those ideas and show interest in promising opportunities.

This platform is developed using React, Node.js, Express, and MongoDB with JWT-based authentication and secure protected routes to manage user access.

A key highlight of this project is that it has been completely developed on an Android mobile device without using a laptop. The entire coding, testing, and development process was done using tools like Acode, Termux, and API testing applications.

## Core Features
- User Registration and Login system
- Secure JWT Authentication
- Protected API Routes
- Entrepreneurs can post business ideas
- Investors can browse and interact with ideas

## Tech Stack
React, Node.js, Express.js, MongoDB, JWT

## Special Highlight
Built entirely on Android mobile using Acode and Termux, without any laptop or desktop support.
